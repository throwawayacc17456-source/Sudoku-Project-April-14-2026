DESCRIPTION:

The project is a sudoku board generator that generates and formats a completed sudoku board.

INSTRUCTIONS:

Copy the MyProgram.java file into a file named as such and do the same for SudokuGenerator.java, then compile

HOW IT WORKS:

The program starts in the first cell and randomly generates a list of numbers to go through, it verifies after each number is plugged in and if it fails it goes back a cell and tries another number.
